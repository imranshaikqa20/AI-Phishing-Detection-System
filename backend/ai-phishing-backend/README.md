# AI Phishing Detection Backend

## Features
- Register API
- Login API
- URL Scan API
- Dashboard API
- PostgreSQL Integration
- Spring Boot Backend

## Run Project

1. Open in IntelliJ
2. Configure PostgreSQL
3. Update application.properties
4. Run BackendApplication.java

## APIs

POST /api/auth/register
POST /api/auth/login
POST /api/scan/url
GET /api/dashboard
